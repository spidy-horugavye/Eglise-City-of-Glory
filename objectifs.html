Nos objectifs

En tant qu’église, nos objectifs sont :

  • Equiper les croyants à travers des formations bibliques solides ;
  • Créer un espace d’adoration authentique ;
  • Soutenir les plus démunis par des actions les plus caritatives ;
  • Encourager la jeunesse dans les valeurs chrétiennes ;
  • Multiplier les églises locales dans plusieurs régions ;
  • Arriver ensemble dans la vie éternelle
